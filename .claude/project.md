# Komi Portfolio - AI-Enhanced Development

## Project Overview
Anime-style portfolio website built with React, Vite, and Framer Motion. Features glassmorphism design, music player with beat detection, Firebase visitor tracking, and responsive design.

## Tech Stack
- **Frontend:** React 18, Vite
- **Animations:** Framer Motion
- **Styling:** CSS3 with glassmorphism
- **Audio:** Web Audio API for beat detection
- **Database:** Firebase Realtime Database
- **Icons:** React Icons
- **Deployment:** Docker + Nginx

## Key Features
- Audio-reactive avatar with rainbow border
- Custom music player with playlist
- Visitor tracking with device fingerprinting
- Responsive mobile/desktop optimization
- Loading screen with click-to-enter
- Project showcase with modal gallery
- Contact section with Discord integration

## Development Guidelines
- Use TypeScript for new components
- Follow glassmorphism design patterns
- Maintain anime aesthetic
- Optimize for mobile-first
- Keep animations smooth (60fps)
- Use semantic HTML
- Implement accessibility features

## File Structure
```
src/
├── components/     # Reusable components
├── hooks/         # Custom React hooks
├── utils/         # Utility functions
├── styles/        # Global styles

```
